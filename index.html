Shortly About the Project

As part of our group project, we installed and set up host websites and chatbots utilizing AWS services. The project's goal is to use Amazon Web Services to build a chatbot and websites (AWS). The actions to accomplish this goal are as follows: selecting AWS services, using HTML and CSS pages to build a website, using different AWS services such as EC2, S3, Route 53 ,and CloudFront

About CIS 4355 Class

This course is intended for students who are interested in learning the fundamentals of cloud computing with regards to planning, deployment, migration and maintenance. Students will also learn about related risks as they pertain to business continuity, security and management
